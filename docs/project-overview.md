# RegTec - Documentació del Projecte

## Resum Executiu

RegTec és un sistema intel·ligent de control de reg que integra sensors IoT i predicció meteorològica per optimitzar l'ús de l'aigua en l'agricultura.

## Objectius del Projecte

1. **Eficiència Hídrica**: Reduir el consum d'aigua fins a un 30%
2. **Sostenibilitat**: Promoure pràctiques agrícoles sostenibles
3. **Automatització**: Minimitzar la intervenció manual en el reg
4. **Predictibilitat**: Utilitzar dades meteorològiques per anticipar necessitats

## Components Tècnics

### Sensors
- Sensors d'humitat del sòl
- Sensors de temperatura ambient
- Sensors de pluviometria
- Sensors de velocitat del vent

### Sistema de Control
- Microcontroladors ESP32
- Comunicació WiFi i LoRaWAN
- Algoritmes d'aprenentatge automàtic
- Interfície web responsive

### Integració Meteorològica
- API de serveis meteorològics
- Processament de dades en temps real
- Prediccions a curt i mitjà termini

## Beneficis Esperats

- Estalvi d'aigua del 20-30%
- Reducció de costos operatius
- Millora de la qualitat dels cultius
- Contribució a la sostenibilitat ambiental
